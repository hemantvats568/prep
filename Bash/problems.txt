Bash practice problems:

1. Word Count
Write a Bash script that takes a sentence as input and counts the occurrence of each word in that sentence. Store the results in an associative array and print the word counts.
Example Input:
bash
Copy code

Example Output:
bash
Copy code
please : 2
pass : 1
sentence : 2
